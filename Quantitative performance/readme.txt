#### This file is a short description for the jupyter notebooks to reproduce the quantitative performance.

To reproduce the quantitative segmentation performance, please run the Quantitative Segmentation_performance_assessment.ipynb in the Jupyter Notebook (python 3).

To reproduce the quantitative detection performance, please run the Quantittive Detection performance assesment.ipynb in the Jupyter Notebook (python 3).
